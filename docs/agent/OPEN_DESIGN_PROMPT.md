# Open Design Prompt

Create a production-grade SaaS visual direction for MiraDeck, a premium AI assistant management platform.

Mission: help users create, configure, and converse with multiple AI assistants from one calm command deck.

Target user: founder, operator, or creator managing personal/productivity assistants.

Primary KPI: a user understands the product from the landing page and can configure + chat with an assistant in under 5 minutes.

Screens/artifacts requested:

1. Public landing page first viewport with brand/logo, CTA, and product preview.
2. Logged-in app shell with assistant library, chat panel, model picker, skills, system prompt editor, memory, files, and image generation surface.
3. Brand token direction: logo, color palette, typography and component style.

Design system:

- Primary inspiration: OpenAI calm editorial AI surface.
- Secondary inspiration: Linear/Vercel dense SaaS polish.
- Avoid: generic neon purple AI glow, bokeh blobs, oversized decorative cards, fake dashboard clutter.

Required states:

- Assistant selected.
- Empty assistant library.
- Chat with thinking mode.
- File uploaded.
- Image prompt generated.
- Profile with avatar/name/bio.

Acceptance criteria:

- Looks like a real 2026 SaaS product.
- Clean, warm, premium, implementable in React/CSS.
- Mobile-safe.
- Brand has own color system, not a copy of OpenAI/Linear.
